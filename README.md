# Machine Learning using Scikit-learn

A comprehensive guide to machine learning with practical projects using **Scikit-learn**, Python's most popular machine learning library.

## 📋 Overview

This project covers fundamental and intermediate machine learning concepts with hands-on implementations. It includes data preprocessing, supervised learning techniques, and real-world predictive modeling projects.

## 📁 Project Structure

### Part-1: Fundamentals & Data Preprocessing
- **data_preprocessing.ipynb** - Essential data preprocessing techniques and transformations
- **Supervised-machine-learning.ipynb** - Introduction to supervised learning algorithms
- **Supervised-machine-learning.py** - Supervised learning implementations
- **Dataset Files**:
  - `airport_disruptions.csv` - Airport disruption data
  - `data.csv` - General dataset

#### Project-1st: Student Performance Prediction
- **first.ipynb** - Initial exploration and baseline models
- **main.ipynb** - Main analysis and model development
- **main.py** - Production-ready code for student performance prediction
- **Student-Prediction_score/** - Advanced student score prediction module
  - `Stu-pred-scor.ipynb` - Detailed notebook for prediction modeling
  - `Student_performance_data_.csv` - Training dataset
- **student_data.csv** - Student information dataset

### Part-2: Advanced Machine Learning
- **Machine-Learning.ipynb** - Advanced ML techniques and applications

#### Student_Success-predictor: Success Prediction System
- **Student_success_predictor.ipynb** - ML pipeline for predicting student success
- **Student_success_dataset.csv** - Comprehensive student success dataset
- **Python utilities** for data processing and model implementation

## 🚀 Quick Start

1. **Prerequisites**
   ```bash
   pip install scikit-learn pandas numpy jupyter
   ```

2. **Explore the notebooks**
   - Start with `Part-1/data_preprocessing.ipynb` for data handling basics
   - Continue with `Part-1/Supervised-machine-learning.ipynb` for learning algorithms
   - Review projects in `Part-1/Project-1st/` for practical applications

3. **Run the predictions**
   ```bash
   python main.py  # For student performance prediction
   jupyter notebook  # To explore interactive notebooks
   ```
